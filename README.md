# gait-legal

Public Terms of Service and Privacy Policy pages for the Gait iOS app.

Live at:
- https://gait-legal.vercel.app/terms
- https://gait-legal.vercel.app/privacy

## Updating

1. Edit the markdown sources in the main app repo: `legal/terms-of-service.md` and `legal/privacy-policy.md`.
2. Run `node build.mjs` here to regenerate `terms.html` and `privacy.html`.
3. Commit and push — Vercel auto-deploys from `main`.
